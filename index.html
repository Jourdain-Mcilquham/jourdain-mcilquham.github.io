Mobility Assessment of Cambridge

R | Python | Social Analysis | Spatial

Using a now defunct social networking application called Gowalla, the aim was to map and understand the mobility patterns of users. The Gowalla dataset allowed users to check-in to locations. Pairing this location based data with Centrality maps creating for Cambridge allowed for the mobility patterns of users to be understood. Using this information, the optimal location for a new venue could be determined.
